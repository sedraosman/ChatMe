# 📱 ChatMe - Flutter Firebase Chat App

A real-time chat application built using **Flutter** and **Firebase**, allowing users to register, log in, and send messages instantly.

---

## 🚀 Features
- 🔐 User Authentication (Sign Up & Login)
- 💬 Real-time messaging with Cloud Firestore
- 👤 Messages displayed based on the sender
- ⚡ Live updates using StreamBuilder
- 🎨 Simple and clean UI

---

## 🛠️ Tech Stack
- **Flutter (Dart)**
- **Firebase Authentication**
- **Cloud Firestore**
